Configure Capture

This the Capture configuration page. The view you see is a preview of what your Capture will look like when you use it to log new entries.

'Configuring' versus 'Adding'

Let’s say you want to start recording how amazing your day was every night before you go to bed. But you don’t want to spend much time on this each night - just a rating from one to ten of awesomeness. Your goal when adding an entry to your Capture is three "taps":

  1. Tap a home screen link to open the Capture to the 'Add Point' page

  2. Tap an icon on a 'star rating' style input to indicate the awesomeness of your day, on a scale of one to 5 stars

  3. Tap 'Save'.

To make this happen we have to configure our Capture. We’d give it a nice name, such as "Today’s Awesome Factor".

You can add a description if you like. Perhaps use the field to describe your own meanings for each star value.

We’ll need two Inputs. The first Input is provided by default - the 'When' value. 'When' simply records the Date and Time. When you’re adding a point you can change the value - for example, if your day was so amazing that you were too exhausted at the end of it to log it, you can log it the next morning and set the value of 'When' to the night before.

You cannot set the value for the When field here in the Configure Capture page - that’s what you’ll do on the 'Add Entry' page. But you can set the default value. The default value is what you will see each time you go to add a new entry for this Capture.

The provided 'When' input is already set to default to 'now' - meaning that when you open the Capture to add a new entry, the 'When' field will be set to the current time.

Our second input is a 'star rating' input type. We’ll need to add that. To do so, we click the add button.

Fields

The Name for your Capture is used throughout the App, while the Description is only displayed here and in some of the Reporting and Output views. Click the edit button to change the Name and Description of this Capture.

'Inputs' describe what you want to log. The simplest Capture (which is often the best Capture!) is just a descriptive Name and a 'When' field. This kind of Capture is wonderfully simple to log: just create a Home Screen shortcut that opens directly to the 'Add Log' page for your capture (don’t worry we describe how to do this on the 'Add Log' page!). Tap the Home Screen shortcut, makes sure you landed in the right Capture and click 'Save'.
